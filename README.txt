Annonce ton JDR est un logiciel qui permet de consulter des annonces de jeux de rôle,
ainsi que de créer des annonces sur toute la France.

Il est disponible pour les systèmes Windows.

Le logiciel propose un onglet de recherche de jeu de rôles en One Shot, où l'on peut régler les paramètres de recherches.
Une fenêtre de création de partie est aussi disponible.
Les résultats sont listés avec les informations les plus nécessaires, classés par ordres chronologiques.
Cliquer sur une ligne de la liste ouvre une page de détails sur la partie. Les phrases sont générées aléatoirement. Une partie peut être édité par le créateur à partir de cette page.

Les informations sont enregistrées dans une base MySQL accessible de partout.
